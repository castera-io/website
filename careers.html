      Sr. UI Developer

      We’re looking for a Senior UI Developer to join our Product Development team and work cross-functionally to generate, test, and implement designs to create a seamlessly navigated experience in quality solutions. Our ideal candidate is a highly motivated, ambitiously drawn to the growth potential, excited to dive in to our fast-paced startup environment.


      WHAT YOU'LL DO

      • You will convert UI design solutions for CAST.ERA products into engineered software.
      • You will interact extensively with UI/UX designers and business team(s) to collaboratively achieve stated business objectives.
      • You will collaborate with other teams and vendors to forge the best technical solutions.
      • You will implement and maintain interface elements, framework, and processes for our standalone and web applications by utilizing continuous integration / continuous delivery (CI/CD) pipeline Jenkins.
      • You will lead UI development through rapid prototype generation, thorough testing, and the integration of features and improvements/fixes, ensuring seamless, functional solutions.
      • You will obtain and incorporate feedback, prioritize features, and demonstrate progress to effectively remain aligned throughout SDLC - development, testing, and deployment phases.


      WHAT YOU BRING

      • 7-10 years of software engineering experience, especially in operator UI development
      • Mastery on HTML5, CSS, JavaScript/TypeScript, node.js, Python / scripting, Go
      • Experience with frameworks in any of the following: Grafana, Angular, JQuery
      • Strong knowledge in Cypress.io, WebPack, Mocha
      • Working knowledge in API and back end development
      • Working knowledge of distributed architectures and strong Object-Oriented (“OO”) fundamentals
      • Expert knowledge of coding for multiple browsers and understanding of their differences and intricate details
      • High proficiency in debugging and fixing UI code, scripts, and backend interface issues (including REST APIs in a distributed environment)
      • Expertise in Atlassian suite of products: Confluence, Jira, and BitBucket
      • Extensive experience in multiple software methodologies, primarily Agile/ Scrum, waterfall, and everything in between
      • Lead development activity using open source software (“OSS”) ecosystem with exposure to a multitude of tools
      • Ability to apply design concepts enough to customize and integrate various OSS-based solutions related to ATSC3.0 broadcasting system

      API Developer

      We’re looking for an API Developer to join and help grow our development team, serving as a primary resource for designing and developing APIs to provide a seamless experience to developers and customers integrating products focused on the broadcasting/ media industry.


      Our ideal candidate is highly motivated, ambitiously drawn to the growth potential, excited to dive into our fast-paced startup environment.


      WHAT YOU'LL DO:

      • Serve as a primary resource for API development. Design, develop, test, and maintain code using open-source tools, software libraries and assume the role of SME in product and API development.
      • Manage/own development of multiple API modules in conjunction with product launch timeline.
      • Develop product adaptors for API standards specific to the broadcasting/media industry (ATSC)
      • Drive API QA best practices, including developing test plans, test descriptions, test procedures, test automation.
      • Perform code reviews and verify the implementation of system and software requirements.
      • Create integration modules and abstract APIs to pass data to and from internal/external systems/products using multiple RESTful APIs.
      • Participate in story grooming/refinement sessions with Product Owners, leveraging platform knowledge to clarify requirements.
      • Work with internal and external teams interfacing with client/customers to understand the products
      • Enforce the API Document to help your colleagues, improve the product, and improve the support experience.
      • Contribute to the ongoing learning and success of the team, and the company, by sharing knowledge through mentorship, collaboration, and aiding in the documentation.


      WHAT YOU BRING

      • Bachelor’s Degree (or equivalent experience) in Engineering, Computer Science, or Information Systems.
      • 7+ years of hands-on software development experience, especially in API implementation and integration,
      • 4+ years of hands-on experience in designing and developing high-volume web services using API Protocols and Data Formats (REST, JSON, SOAP & XML).
      • Experience with various Web APIs (client and server), knowledge about current API frameworks. Technologies like REST, GraphQL, gRPC, and opensource based software development.
      • Hands-on experience in developing and testing (automation) using the APIGEE management platform, including but not limited to installation and configuration.
      • Implement requirements of the API layer like security, custom analytics, throttling, caching, logging, request, and response modifications, etc. using the API management platform.
      • Mastery understanding of modern application architectures - three-tier, distributed, centralized, microservices, etc., with experience in REST API.
      • Expert level experience with some or all the following technologies: Java, Go, Python 3, SpringBoot, Web sockets, Git, Mochas, and public/private cloud.
      • Proficient in Linux/Open-Source technologies.
      • Experience with central logging and dashboarding systems like Elasticsearch, Fluentbit, Filetbit, Kibana, Grafana, Graylog.
      • Experience/strong willingness to learn multimedia and/or broadcasting/ media applications and technologies is highly desired.
      • General understanding of database products such as MS SQL, MySQL, PostgreSQL, MongoDB, Cassandra, Elasticsearch.
      • Experience in the broadcast or OTT industry is a huge plus.

      Our ideal candidate is highly motivated, ambitiously drawn to the growth potential, excited to dive into our fast-paced startup environment.

      WHAT YOU'LL DO:

      • Serve as a primary resource for DevOps development and maintenance.
      • Design, develop, test, and maintain pipeline code using open-source tools, software libraries and assume the role of SME in DevOps Platform.
      • Work collaboratively with software engineering and management to define infrastructure and deployment requirements.
      • Follow the industry best practice and continuously improve the CI-CD efficiency and security.
      • Provides and manages test automation and performance testing capabilities to validate applications and the environment.
      • Responsible for establishing, operating, and maintaining continuous integration, continuous delivery, and automated testing frameworks.
